# 🛒 CTK SHOP

## Opis projekta

CTK SHOP je web aplikacija za upravljanje trgovinama i artiklima. Korisnici se mogu registrirati, prijaviti, kreirati trgovine i dodavati artikle.

## 🧑‍💻 Tehnologije

- **React** 
- **TypeScript** 
- **Bulma** 
- **Sass (SCSS)** 
- **Axios** 
- **React Router** 
- **json-server** 
- **Git + GitHub**
